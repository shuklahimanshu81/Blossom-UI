Avatars

Avatars are used in profile pictures. I have listed 3 kinds of avatars that you can use in you code.

Rounded image avatar

To use round image avatar in your project add round-avatar class to the element followed by variant of avatar.

Square imgae avatar

To use square image avatar in your project add square-avatar class to the element followed by variant of avatar.

Text avatar

To use Text avatar in your project add text-avatar class to the element followed by variant of avatar.
